# 서버리스

## 서버리스 소개

- IaaS(Infra as a Service)
- PaaS(Platform as a Service)

IaaS -> PaaS

- FaaS(Function as a Service)

FaaS는 Serverless의 부분집합

서버리스의 장점

- 인프라관리 감소
- 확장성
- 빠른 output
- 경제력

서버리스의 단점

- 낮은 호환성
- 높은 종속성
- 한정적
- 무상태

서버리스에 적합한 시스템

- 스케줄러
- IoT
- 배치
- 예측 불가
- 작고, 빠르고, 간단한 것
- API
